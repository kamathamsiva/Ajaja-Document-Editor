# AI Workflow

## Overview
This project is an AI-assisted collaborative document editor built using the MERN-style stack with React, Node.js, Express, and SQLite.

## Development Workflow

1. Designed the project architecture.
2. Built the backend REST APIs using Express.js.
3. Created the SQLite database and tables.
4. Implemented JWT-based authentication.
5. Developed the React frontend.
6. Integrated frontend and backend using Axios.
7. Added document creation, editing, saving, and sharing.
8. Tested APIs using Postman.
9. Fixed bugs and improved the UI.
10. Deployed the final source code to GitHub.

## AI Assistance
AI was used to:
- Generate boilerplate code.
- Explain errors.
- Suggest best practices.
- Debug backend and frontend issues.
- Improve project documentation.
